================================================================================
                           HARDWARE REQUIREMENTS
================================================================================

ARDUINO SETUP:
--------------
- Arduino Uno/Nano/Mega
- DHT22 Temperature & Humidity Sensor
- BH1750 Light Sensor (I2C)
- Analog pH Sensor (connected to A0)
- Soil Moisture Sensor (connected to A1)
- USB Cable (for serial communication)

WIRING:
-------
DHT22:
  - VCC -> 5V
  - GND -> GND
  - DATA -> Digital Pin 2

BH1750 (I2C):
  - VCC -> 5V
  - GND -> GND
  - SDA -> A4
  - SCL -> A5

pH Sensor:
  - VCC -> 5V
  - GND -> GND
  - Signal -> A0

Soil Moisture:
  - VCC -> 5V
  - GND -> GND
  - Signal -> A1

================================================================================
                          STEP-BY-STEP SETUP
================================================================================

STEP 1: Clone/Copy the Project
------------------------------
Copy the entire `minor-project` folder to your new laptop.

STEP 2: Install Node.js Dependencies
-------------------------------------
Open terminal in the project folder:

    cd "/path/to/minor-project"
    npm install

This installs all app dependencies from package.json.

STEP 3: Install Server Dependencies
------------------------------------
    npm install express cors serialport

STEP 4: Configure Environment Variables
----------------------------------------
    cp .env.example .env

Edit `.env` and add your API keys:
- Get OpenRouter key from: https://openrouter.ai/keys
- Get Supabase credentials from: https://supabase.com/dashboard

STEP 5: Upload Arduino Code
---------------------------
1. Open Arduino IDE
2. Open: arduino/SmartAgri_Final.ino
3. Install required libraries (see ARDUINO LIBRARIES section)
4. Select your board (Tools > Board > Arduino Uno)
5. Select your port (Tools > Port)
6. Click Upload

STEP 6: Configure Server Port
-----------------------------
Edit `server.js` and update SERIAL_PORT:

    Windows: 'COM3' or 'COM4' (check Device Manager)
    Linux:   '/dev/ttyUSB0' or '/dev/ttyACM0'
    Mac:     '/dev/tty.usbmodem14101' or similar

STEP 7: Find Your Computer's IP Address
----------------------------------------
    Windows: ipconfig (look for IPv4 Address)
    Linux:   hostname -I
    Mac:     ifconfig | grep inet

STEP 8: Update App IP Address
-----------------------------
Edit `src/screens/DashboardScreen.js`:
Change line 15:
    const SENSOR_SERVER = 'http://YOUR_IP_ADDRESS:3001';

Example:
    const SENSOR_SERVER = 'http://192.168.1.101:3001';

STEP 9: Start the Sensor Server
-------------------------------
In a terminal:

    node server.js

You should see:
    =====================================
       SENSOR DATA SERVER
    =====================================
    Server running on http://localhost:3001
    Arduino connected on /dev/ttyUSB0

STEP 10: Start the Expo App
---------------------------
In another terminal:

    npx expo start

Scan the QR code with Expo Go app on your phone.
Make sure phone and laptop are on the same WiFi network!

================================================================================
                           TROUBLESHOOTING
================================================================================

ERROR: "Cannot find module 'express'"
FIX: Run `npm install express cors serialport`

ERROR: "Cannot open COM3" or serial port errors
FIX: Check Arduino port in Arduino IDE (Tools > Port)
     Update SERIAL_PORT in server.js

ERROR: "Network request failed" in app
FIX: Verify IP address is correct
     Ensure phone and laptop are on same WiFi
     Check firewall isn't blocking port 3001

ERROR: "Missing environment variables"
FIX: Create .env file from .env.example
     Add your API keys
     Restart Expo with `npx expo start --clear`

ERROR: pH value showing wrong number
FIX: Calibrate pH sensor with buffer solution
     Adjust voltage_pH7 value in Arduino code
